about

Gardenlink es un sistema de control de invernaderos de codigo abierto, que nace a partir de un experimento para controlar de forma remota un cultivo de hortalizas. Esta compuesto de 4 modulos:

  • GardenLink.API: Servicios REST para interactuar con el invernadero
  • GardenLink.MQTTServer: Broker para conectar la API con Arduino
  • (TBD) GardenLink.ArduinoClient: Sketch para controlar sensores y actuadores
  • GardenLink.Web: Mantenedores y Paneles de Control
